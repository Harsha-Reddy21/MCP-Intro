# MCP Knowledge Base - AI Assistant


### 1. Backend Setup
```bash
cd backend
python -m venv myenv
pip install fastapi uvicorn scikit-learn numpy requests

```

### 2. Frontend Setup
```bash
cd ../frontend

npm install
npm start
```



## 🌐 Access the Application

- **Frontend (React):** http://localhost:3000
- **Backend API:** http://localhost:8000
